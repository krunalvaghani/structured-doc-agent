Structured Doc Agent

Upload a PDF, define fields, watch the agent work, get JSON.

How this works
  • Pipeline — validate file → build JSON Schema → agent reads the document → structured JSON out.
  • Tools first — the model analyzes the file, then reads selectable text or renders page images (scanned PDFs).
  • Honest output — values come from the document only; missing fields are null, not guessed.
  • Smart models — if a scanned doc needs vision and your model is text-only, a vision model is picked automatically (shown in the feed).
  • Observable — event history, live status, and run summary show what happened behind the scenes.
